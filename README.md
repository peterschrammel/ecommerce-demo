# E-commerce Demo

Demo of a Java e-commerce web site using Spring Framework.

Requirements:
- Maven
- Java 8

To build type:
  `mvn compile`

To test type:
  `mvn test`

To run type:
  `mvn spring-boot:run`

When project is running, navigate to:
 `localhost:8080`
